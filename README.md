# Todo Manager

A simple command-line task manager written in C++.

## Features

- Add, view, and delete tasks
- Tasks are stored in `tasks.txt`
- Simple menu-driven interface
- Save tasks between program runs

## Requirements

- C++17 compatible compiler (e.g., g++)
- Make utility

## How to Build

```bash
# From the project root
make
# Run the program
./bin/program
# or via Makefile
make run
# to clean the build files type
make clean

#for now:
Usage:

Option 1: Show all tasks
